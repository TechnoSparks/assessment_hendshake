## Getting Started

1. Clone the repo
```bash
git clone https://github.com/TechnoSparks/assessment_hendshake.git
cd repo-name
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Run the dev environment
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
